## Deskripsi Program
dengan memanfaatkanProgram Sistem Item AIC (Automated Industry Complex) ini menerapkan sistem list produk berbasis **Java** yang dibuat menggunakan **INTELLIJ IDE**  struktur data **Arraylist**.

Disini kita membuat program CRUD (Create, Read, Update, Delete)
yang berfungsi untuk menyimpan data produk yang ada di AIC ini.
Disini kita menerapkan Abstract dan Interface.

## Konsep Pemrograman
### Abstract Class dan Method
Pada abstract class terdapat `produkAIC` yang bertindak sebagai template hierarki sedangkan pada abstract method terdapat `DetailCategory()` yang memastikan setiap subclass untuk menjelaskan diri mereka sendiri.

### Interface
`Dijiang` disini digunakan sebagai kontrak tambahan untuk fitur audit atau status yang tidak dimiliki semua produk.
